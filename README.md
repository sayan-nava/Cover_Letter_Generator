# Cover Letter Generator

Simple Python Program to generate Cover Letter

## Description

Saves you from the hassle of editing Cover Letters during your job hunt

## Getting Started

### Dependencies

* Python3
* Jinja2

### Installation
Before proceeding, please make sure that:
* You edit all the necessary details in template.html(currently commented out)
* Python and Jinja is installed in your system
```
pip install Jinja2
```

### Executing program
```
python main.py
```
* Enter the next company's name and designation
* Cover_Letter.txt will be genarated in few seconds
